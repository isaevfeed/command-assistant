# Описание

Библиотека для упращения вызова команд для работы с файлами

- указываем проект
- указываем пути до стендов
- указываем саму команду и параметры (arguments)

### Формат файла (примерный)

```
{
    "tn": {
        "decode": {
            "command": "sops",
            "arguments": ["-d", "-i"]
        },
        "encode": {
            "command": "sops",
            "arguments": ["-e", "-i", "--pgp", "\"fingerprint\""]
        },
        "prod": "/Users/...",
        "test": "/Users/...",
        "stage": "/Users/..."
    }
}
```
